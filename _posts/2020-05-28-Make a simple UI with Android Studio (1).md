## Android Studio (1)

Let's create a simple User Interface(UI) with Android Studio.

I took <a href="#" title="https://developer.android.com/training/basics/firstapp">https://developer.android.com/training/basics/firstapp</a> as a reference. You can check out the website for more details.

There are four main steps:
<ol>
<li>Add a text box</li>
<li>Add a button</li>
<li>Change the UI strings</li>
<li>Make the box size flexible</li>
</ol>

In an Android app, the user interface(UI) is composed as an hierarchy of <i><b>layouts</b></i> (=ViewGroup objects) and <i><b>widgets</b></i> (=View objects).

<ol>
<li><b>Layouts/ViewGroup</b></li>
    -<i>containers</i> that control how their child views are positioned on the screen
<li><b>Widgets/View</b></li>
    -<i>UI components</i> (i.e. buttons and text boxes)
</ol>
